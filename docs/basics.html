Buttons

Button styles are applied to the BUTTON element. The class ".btn-change" provides a different back color.

<button type="button">&times;</button>
<button type="submit">Save</button>
<button type="reset" class="btn-change">Clear</button>
<button type="button" class="btn-change">Edit</button>
<button type="button" disabled>Disabled</button>

Fieldset

Any FIELDSET element and LEGEND will be styled as follows:

Legend Fieldset content 
    <fieldset><legend>Legend</legend>
      Fieldset content
    </fieldset>
